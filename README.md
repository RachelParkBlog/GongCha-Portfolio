## 적용 기술 및 구현 기능

### 적용 기술

> - Front-End : HTML5, CSS3, JavaScript, jQuery
> - Back-End : PHP, MySQL
> - Common : Photoshop, Illustrator, Storyboard

### 구현 기능

> Sign Up/In

- 카카오 API 연동을 통한 소셜 로그인 구현

> List Page

  - 상단 카테고리 필터링 기능을 통해 카테코리 click시 
    해당 페이지로 이동 기능 구현

- 이 프로젝트에서 사용하고 있는 제품 사진의 저작권은 전적으로 공차 코리아에 있습니다.
